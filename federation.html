Stellar Federation for DEV Token

We've enabled federation so you can send DEV using easy-to-read Stellar addresses like:

team*stellarport.pages.dev

How it works

Instead of using long account IDs, you can use a name like:

  • team*stellarport.pages.dev → Issuer Account
  • grant*stellarport.pages.dev → Distribution Account

Test it

Try querying:

https://stellarport.pages.dev/federation?q=team*stellarport.pages.dev&type=name

Supported Accounts

  • team* → Developer Foundation / Issuer
  • grant* → Distribution + Grants
